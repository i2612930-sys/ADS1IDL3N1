¿Qué es HTML?

El lenguaje de marcado de hipertexto (HTML) permite estructurar y desplegar el contenido de una página web, para ello se basa en un conjunto de elementos identificados con etiquetas, los más comunes son:

  • Encabezados
  • Párrafos
  • Listas
  • Imágenes
  • Vínculos
  • Frames

Estructura de Programacion

La estructura de programación web se compone de múltiples elementos interactuando en conjunto. Para desarrollar un sitio completo es fundamental dominar el uso correcto de:

  • Párrafos: Para organizar textos informativos y lecturas ordenadas.
  • Listas: Ya sean ordenadas o desordenadas para estructurar datos.
  • Imágenes: Elementos gráficos indispensables para la experiencia visual.
  • Vínculos: Enlaces esenciales que conectan las páginas de toda la red.
